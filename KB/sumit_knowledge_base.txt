# PROJECT: Nepali Grammar Error Correction System

Overview:
I built a grammar error correction system for Nepali using transformer models.

Problem:
Nepali is a low-resource language, making it difficult to build high-quality NLP systems.

Solution:
I fine-tuned a multilingual T5 model using synthetic parallel data and LLM-generated corrections.
I also trained a BERT-based reranker to improve output quality.

Tech Stack:
PyTorch, Hugging Face Transformers, FastAPI

Outcome:
The reranking approach improved correction quality and overall system performance.
